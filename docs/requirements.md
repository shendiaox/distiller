# Distiller（万物可蒸馏）— 需求文档

> 版本：v1.0 | 日期：2026-05-30 | 状态：已确认

---

## 一、项目概述

**Distiller（万物可蒸馏）** 是一款 Windows 桌面知识蒸馏工具。用户可以将某个人物、某个领域的资料录入知识库，通过 AI 提炼出其中的**做事方法论、思维模式、话术风格**等模式，并以聊天方式与蒸馏结果互动。

**目标用户**：需要深度研究人物/领域知识、提取可复用方法论的个人用户。

---

## 二、核心功能

| 编号 | 功能 | 详细说明 | 优先级 |
|------|------|----------|--------|
| F1 | 文本录入 | 支持直接粘贴文本、上传 .txt 文件到知识库 | P0 |
| F2 | 向量化存储 | 文本自动分块、向量化，支持语义检索 | P0 |
| F3 | 知识库问答 | 基于知识库内容的对话式问答（RAG） | P0 |
| F4 | 人物蒸馏 | 从知识库中提炼人物的做事方法论、思维模式、话术风格 | P0 |

## 三、扩展功能

| 编号 | 功能 | 详细说明 | 优先级 |
|------|------|----------|--------|
| F5 | 文件上传 | 支持上传 .docx / .pdf 文件自动解析录入 | P1 |
| F6 | 全网搜索 | 输入人名/关键词，自动搜索并填充到知识库 | P1 |
| F7 | Skills 模板 | 预设分析模板（如"商业模式分析""决策模式提取"） | P1 |
| F8 | Skills 角色 | 可保存的定制 AI 角色（如"马斯克商业顾问"） | P1 |
| F9 | 风格生成 | 基于某创作者的风格生成新内容（如方文山风格歌词） | P2 |
| F10 | 多知识库 | 创建多个独立知识库，切换使用 | P2 |
| F11 | 知识库导出 | 知识库内容导出为文件 | P2 |

## 四、非功能需求

- **平台**：Windows 10/11
- **离线能力**：向量检索本地完成，AI 对话需联网（DeepSeek API）
- **隐私**：所有资料本地存储，不上传至第三方（DeepSeek API 调用除外）
- **性能**：万级文本块下检索延迟 < 2 秒
- **安全**：API Key 本地加密存储

## 五、三大使用场景详述

### 场景 A：人物蒸馏
> "我想知道马斯克在商业领域的做事方法"

1. 全网搜索马斯克相关内容 → 自动录入知识库
2. 用户补充上传马斯克传记、访谈 PDF
3. 选择"商业方法论蒸馏"模板
4. AI 输出：马斯克的核心决策原则、风险偏好、团队管理方式、沟通风格等
5. 用户可以追问、深挖具体方面

### 场景 B：领域专家
> "我上传了 3 本肿瘤学教材，帮我理解 TNM 分期"

1. 用户上传专业教材 PDF
2. AI 基于教材内容回答专业问题
3. 引用来源章节，避免幻觉

### 场景 C：风格生成
> "我把方文山所有歌词给你，你给我生成一首方文山风格的中国风歌词"

1. 用户录入方文山歌词全集
2. 选择"风格蒸馏"模板
3. AI 提炼方文山的词汇偏好、意象体系、韵脚模式
4. 用户给出主题要求，AI 生成符合风格的新歌词

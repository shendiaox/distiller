# Distiller（万物可蒸馏）— 设计规范

> 版本：v1.0 | 日期：2026-05-30

---

## 一、设计风格

**暗色极简 + 暖色点缀**：类似 ChatGPT/Claude 的对话界面风格，但使用更暖的暗色调，营造"书房/蒸馏室"的氛围感。

- 深色背景，减少视觉疲劳（长时间阅读/对话场景）
- 暖金色点缀，呼应"蒸馏"的化学意象
- 信息密度中等，对话内容清晰可读
- 侧边栏收纳知识库和 Skills 管理

## 三、布局结构

```
┌──────────────────────────────────────────────────────────┐
│  标题栏：Distiller Logo + 当前知识库名称 + 窗口控制        │
├────────────┬─────────────────────────────────────────────┤
│  侧边栏    │                                             │
│            │          ┌─────────────────────┐            │
│  📁 知识库  │          │                     │            │
│  ├ 马斯克   │          │   对话消息流         │            │
│  ├ 肿瘤学   │          │   AI 回答           │            │
│  └ + 新建   │          │   用户问题           │            │
│            │          │   AI 回答（流式）     │            │
│  🎯 Skills  │          │                     │            │
│  ├ 商业分析  │          │                     │            │
│  ├ 风格克隆  │          │                     │            │
│  └ + 新建   │          │                     │            │
│            │          └─────────────────────┘            │
│            │  ┌──────────────────────────────────────┐   │
│  知识库内容 │  │  输入框                        发送  │   │
│  📄 文档列表 │  └──────────────────────────────────────┘   │
│  🔍 全网搜索 │                                             │
│  📎 上传文件 │                                             │
└────────────┴─────────────────────────────────────────────┘
```

- **左侧栏（260px）**：知识库切换、Skills 列表
- **右侧主区**：对话流 + 底部输入框
- **最小窗口**：1280×720
- **默认大小**：1400×900

## 五、交互规范

| 操作 | 行为 |
|------|------|
| 发送消息 | Enter 发送，Shift+Enter 换行 |
| 新建知识库 | 点击侧边栏"+ 新建"，弹窗输入名称 |
| 切换知识库 | 点击侧边栏知识库名，切换对话上下文 |
| 录入内容 | 点击"粘贴文本"打开文本框 / 拖拽文件上传 |
| 全网搜索 | 点击"全网搜索"，输入关键词，自动搜索+录入 |
| 选择 Skill | 点击侧边栏 Skill，注入当前对话 |
| 蒸馏操作 | 输入框上方选择蒸馏模式，触发 Prompt Chain |

---

## 六、窗口规格

- 默认分辨率：1400×900
- 窗口模式（非全屏，适合多任务并行）
- 可调整大小，最小 1280×720
- 启动居中显示
